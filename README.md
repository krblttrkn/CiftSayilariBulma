## While Döngüsü ile Çift Sayıları Bulan Program 

```
import java.util.Scanner;

public class denden {
    public static void main(String[] args){
        int i=1,k;
        Scanner input=new Scanner(System.in);
        System.out.print("Bir Sayı Giriniz : ");
        k=input.nextInt();

        while(i<=k){

            if (i%2==0){
                System.out.print(i+",");
            }
            i++;
        }
    }
}
```
